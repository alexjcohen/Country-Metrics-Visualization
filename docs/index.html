Notes About The Data

The data being looked at is from 2010-2014/2015. The reason for this is that more recent years (e.g. 2016+) do not have enough data to reliably produce charts for specific metrics, such as Education and Healthcare expenditures. Unfortunately, even when reducing the data to these few years, there are several missing values from the dataset that will be excluded on the following pages.
